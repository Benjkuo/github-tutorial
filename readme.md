## github指令
- config
    >git config --global user.name myusername   
    >git config --global user.email myemail  
    >git config --global github.user myusername  
- clone from others
    1. git clone {url}
- upload to yours
    1. git status
    2. git add *
    3. git commit -m 'message'
    4. git push
    5. checkout your github
